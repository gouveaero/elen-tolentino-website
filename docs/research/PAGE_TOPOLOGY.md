# Page Topology — elentolentino.com.br

## Design System Summary
- **Font**: Inter (400, 500, 700, 800) via Google Fonts
- **Primary pink/magenta**: `#E10098` = `rgb(225, 0, 152)`
- **Hot pink CTA (Pós-Grad)**: `#FF0074` = `rgb(255, 0, 116)`
- **Deep rose (EstomatoPlay bg)**: `#CC3366` = `rgb(204, 51, 102)` / `#571A31` = `rgb(87, 26, 49)`
- **Dark BG (Home)**: `#121212` = `rgb(18, 18, 18)` / `#1E1E1E` = `rgb(30, 30, 30)`
- **Dark text (light pages)**: `#333333` = `rgb(51, 51, 51)`
- **Light pink bg (Pós-Grad)**: `#FFE7F5` = `rgb(255, 231, 245)`
- **White**: `#ffffff`
- **Gray-200**: `#e5e7eb` (Tailwind default, used for borders)
